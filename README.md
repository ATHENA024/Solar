# Solar System Explorer

An interactive solar system explorer built entirely with CSS — no JavaScript required.

## Features

- 3D solar system visualization with CSS transforms
- Clickable planet menu with detailed information panels
- Planet and moon orbital trajectories
- Pure CSS animations and transitions
- Responsive layout

## Usage

Open `index45.html` in any modern browser. Click on a planet name from the left menu to explore it in detail. Read more about each planet by clicking the "Read More" link.

## Tech Stack

- HTML5
- CSS3 (transforms, animations, 3D perspective)
- Google Fonts (Montserrat)
